1) SISTEMAZIONE GOOGLE SHEET / APPS SCRIPT
- Apri il foglio Google.
- Vai su Estensioni > Apps Script.
- In Code.gs incolla il file apps_script_DoseQuest_Pediatrico_CONFIGURATO_foglio_Cristina.gs.
- Salva.
- Nel menu funzioni scegli setup.
- Clicca Esegui.
- Autorizza con il tuo account Google.
- Se compare app non verificata: Avanzate > Vai al progetto > Consenti.
- Poi Distribuisci > Gestisci distribuzioni.
- Verifica che la Web App sia distribuita come:
  Esegui come: Me
  Chi ha accesso: Chiunque abbia il link / Chiunque / Chiunque abbia un account Google.

2) PUBBLICAZIONE SU GITHUB PAGES
- Crea una repository GitHub, per esempio: dose-quest-pediatrico.
- Carica nella root della repository questi file:
  index.html
  .nojekyll
- Non rinominare index.html: GitHub Pages lo apre automaticamente.
- Vai su Settings > Pages.
- Source: Deploy from a branch.
- Branch: main / root.
- Clicca Save.
- Aspetta qualche minuto.
- GitHub ti darà un link del tipo:
  https://TUO-NOME.github.io/dose-quest-pediatrico/

3) QR CODE
- Apri il link pubblico GitHub Pages del quiz.
- Clicca Docente / Google.
- Nel campo URL pubblico del quiz per QR clicca Usa URL attuale.
- Clicca Genera QR.
- Stampa o proietta il QR in aula.

4) COME FUNZIONA L'INVIO AUTOMATICO
- Lo studente compila la registrazione iniziale.
- Fa il quiz.
- Clicca Completa.
- Il quiz invia automaticamente i dati alla Web App Google Script già inserita.
- I risultati entrano nel Google Sheet nei fogli:
  Risultati Quiz
  Dettaglio Risposte

5) TEST CONSIGLIATO
- Prima di usarlo in aula, apri il quiz da GitHub Pages.
- Inserisci Nome: TEST Cristina.
- Rispondi a qualche domanda.
- Clicca Completa.
- Controlla il Google Sheet.

NOTA IMPORTANTE
Il QR non deve puntare al file locale del computer. Deve puntare al link GitHub Pages pubblico.
